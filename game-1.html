JavaScript Fighters

Enter the world of javascript and clash with your friends in JavaScript Fighters! This is a local 2-player fighting game you can enjoy with your friends on the same keyboard.
Notes: The game does not currently restart automatically; to play again refresh the page.

Controls:
Player 1: W = Jump, A = Left, D = Right, Space = Attack
Player 2: Up = Jump, Left = Left, Right = Right, Down = Attack

Developer: Bennie D. Cooper
